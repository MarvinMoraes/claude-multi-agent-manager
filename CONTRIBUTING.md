# Guia de Contribuição

Obrigado por considerar contribuir para o **Claude Multi-Agent Manager**!

## Como Contribuir

1. **Fork** o repositório
2. **Crie uma branch** descritiva: `git checkout -b feature/nome-da-feature`
3. **Faça commit** com mensagens claras seguindo [Conventional Commits](https://www.conventionalcommits.org/pt-br/):
   - `feat:` nova funcionalidade
   - `fix:` correção de bug
   - `docs:` documentação
   - `chore:` tarefas de manutenção
   - `test:` adição de testes
4. **Push** para sua branch: `git push origin feature/nome-da-feature`
5. Abra um **Pull Request** detalhando as mudanças

## Padrões de Código

- Use **type hints** em todas as funções
- Escreva **docstrings** para classes e métodos públicos
- Adicione **testes** para novas funcionalidades
- Siga o estilo **PEP 8**

## Reportando Bugs

Abra uma [Issue](https://github.com/MarvinMoraes/claude-multi-agent-manager/issues) com:
- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs. observado
- Versão do Python e das dependências
